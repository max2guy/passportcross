Part 2 — 설치 · 알림
📲
앱 설치 방법 (PWA)
▼

이 앱은 아이폰(Safari)과 안드로이드(Chrome)에서 '홈 화면에 추가'를 통해 실제 어플처럼 설치할 수 있습니다. 가급적 설치 후 사용해 주세요.

i
아이폰 (Safari) — 하단 공유 버튼(□↑) 탭 → '홈 화면에 추가' 선택
A
안드로이드 (Chrome) — 우측 상단 메뉴(⋮) 탭 → '홈 화면에 추가' 또는 설치 배너 탭

Part 3 — 교사 · 임원 (관리자)
1
관리자 패널 진입
▼
  • 일반 학생과 동일한 링크로 접속합니다.
  • [더보기] 탭에서 교사 이메일/비밀번호로 로그인하면 숨겨진 관리자 메뉴가 열립니다.
  • 계정이 없다면 초대 코드 입력 후 직접 가입할 수 있습니다.
🔑
초대 코드 필요
관리자 가입 시 마스터 계정에서 발급된 초대 코드가 필요합니다. 담당 교역자에게 문의하세요.
2
미션 승인 및 반려
▼
  • [대기중인 제출] 목록에서 학생들의 미션을 확인합니다.
  • 승인(초록) 처리 시 학생에게 즉시 푸시 알림이 발송되고 여권에 도장이 찍힙니다.
  • 반려(빨강) 처리 시 사유를 입력하면 학생이 다시 제출할 수 있습니다.
✓ 승인
✕ 반려 + 사유 입력
3
미션 잠금 및 편집
▼
  • 날짜에 상관없이 강제로 미션을 열거나 잠글 수 있습니다.
  • 예: 주말에 밀린 미션 허용, 특정 미션 일시 잠금
  • 앱 내에서 직접 미션 내용과 인증 방식을 수정할 수 있습니다.
⚙ 자동 (날짜 기반) 🔓 강제 열림 🔒 강제 잠금
4
최고 관리자 전용 (마스터 계정)
▼
  • 교사 계정 추가 / 삭제 및 초대 코드 발급
  • 전체 학생 데이터 초기화 등 핵심 권한은 마스터 계정에만 부여됩니다.
