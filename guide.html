01. 프로그램 개요

✝ 무엇인가요?

THE CROSS PASSPORT는 고난주간(성월요일~성토요일) 6일 동안 매일 하나의 미션을 실천하고, 사진·글로 인증하는 모바일 앱 기반 제자훈련 프로그램입니다.

학생들은 앱을 스마트폰 홈화면에 설치하고, 미션을 완수할 때마다 '여권 도장'을 받아 6일을 완주하면 부활주일에 수료증과 선물을 받습니다.

🎯
목회적 의도 "THE CROSS PASSPORT는 자격증이 아닙니다." 미션을 수행하는 이유는 하나님께 점수를 따기 위함이 아닙니다. 우리를 위해 모든 물과 피를 쏟으신 그리스도의 은혜에 감사하여, 우리 삶의 주인이 누구인지 고백하는 작은 응답입니다.

Student Guide

03. 학생 앱 사용 방법

1
앱 설치 (홈화면에 추가)

담당 교사에게 앱 주소(QR코드)를 받아 Safari(iOS) 또는 Chrome(Android)으로 접속한 뒤, 공유 버튼 → "홈 화면에 추가"를 탭합니다. 일반 앱처럼 홈화면 아이콘이 생깁니다.

2
이름·학년·그룹 등록 (최초 1회)

앱 첫 실행 시 이름, 학년(또는 직장), 소속(중고등부/청년부)을 입력합니다. 이 정보는 관리자 화면에 그대로 표시됩니다.

3
매일 미션 탭에서 당일 미션 확인

하단 [미션] 탭을 누르면 6일치 미션 카드가 보입니다. 오늘 날짜 미션만 열리며, 이후 날짜는 잠겨 있습니다. 카드를 탭하면 성구, 내용, 인증 방법이 표시됩니다.

4
미션 실천 후 인증 제출

사진 미션은 📷 사진 첨부 후 [실천 완료] 버튼을 누릅니다. 텍스트 미션은 소감을 입력 후 제출합니다. 화요일 미션(부모 확인)은 교사가 PIN을 입력해 승인합니다.

5
관리자 검토 후 도장 획득

제출 즉시 [대기중] 상태가 되며, 관리자가 확인 후 승인하면 [여권] 탭에 도장이 찍힙니다. 앱이 켜져 있으면 실시간으로 알림이 옵니다.

6
6일 완주 → 수료증 & 선물 신청

6개 도장을 모두 받으면 [더보기] 탭 → [수료증 보기]에서 수료증을 이미지로 저장할 수 있습니다. 수료증 화면에서 [수료선물 받기] 버튼으로 선물을 신청합니다.

💡
배경화면 & 배지 [배경화면] 탭에서 매일 테마에 맞는 잠금화면 배경화면을 저장할 수 있습니다. 미디어 금식 배지도 프로필 이미지로 사용 가능합니다.
Status System

04. 미션 상태 시스템

미완료
아직 제출하지 않은 상태 오늘 날짜 미션만 열려 있으며, 미래 날짜는 자동으로 잠겨 있습니다.
대기중
학생이 제출 완료 → 관리자 검토 대기 [미션] 탭 카드에 "검토 중" 표시. 관리자 앱에 즉시 목록이 올라옵니다.
승인
관리자가 승인 → 여권 도장 획득 [여권] 탭에 도장이 찍히고 학생 앱에 실시간으로 반영됩니다.
반려
관리자가 반려 → 재제출 가능 반려 사유가 앱에 표시되며, 학생은 수정 후 다시 제출할 수 있습니다.
Admin Guide

05. 관리자 앱 사용 방법

🔐
관리자 계정 필요 관리자 앱은 이메일/비밀번호로 로그인합니다. 계정은 담당 전도사·목사님께 받으세요. 일반 학생이 접근할 수 없습니다.
① 로그인
관리자 앱 주소에 접속 → 이메일·비밀번호 입력 → 로그인
② 제출 목록 확인
상단 통계(총 제출 / 대기 / 승인 / 반려)와 필터(그룹별·상태별)로 목록을 확인합니다. 사진 첨부가 있으면 썸네일이 표시됩니다.
③ 승인 또는 반려
제출 카드에서 [✔ 승인] 또는 [✘ 반려]를 누릅니다. 반려 시 사유를 입력하면 학생 앱에 바로 전달됩니다.
④ 수료선물 신청자 확인
[수료선물] 탭에서 6일 모두 완주한 학생 목록을 확인합니다. 이름·학년·그룹·신청 시각이 표시됩니다.
⑤ 미션 잠금/해제 (선택)
기본값은 날짜 기준 자동 잠금입니다. 필요 시 관리자 앱에서 특정 미션을 강제 열기/닫기할 수 있습니다.
📌
화요일 미션 (부모/교사 PIN 승인) 중고등부 화요일 미션 "디지털 효도"는 부모님 또는 교사가 직접 학생 폰에 4자리 PIN을 입력해 승인하는 방식입니다. PIN은 관리자 앱 로그인 시 표시되는 번호를 사용하거나 담당자에게 별도 안내 받으세요.

FAQ

07. 자주 묻는 질문

Q. 앱을 설치해야 하나요? 브라우저에서도 되나요?

브라우저에서도 작동하지만, 홈화면 설치를 권장합니다. 설치하면 오프라인에서도 기본 기능이 동작하고, 앱처럼 전체화면으로 사용할 수 있습니다.

Q. 폰을 바꾸거나 앱을 지웠어요. 기록이 사라지나요?

인터넷이 연결된 상태에서 앱을 사용했다면 Firebase 클라우드에 자동 저장됩니다. 같은 기기에서 앱을 재설치하면 기록이 복구됩니다. 단, 기기가 완전히 바뀐 경우 담당 관리자에게 문의하세요.

Q. 미션을 잘못 제출했어요.

관리자가 반려 처리 후 사유를 전달하면, 학생은 수정 후 재제출할 수 있습니다. 관리자에게 직접 문의하는 것이 가장 빠릅니다.

Q. 오늘 날짜 미션이 잠겨 있어요.

기기 날짜/시간이 잘못 설정되어 있을 수 있습니다. 설정 → 일반 → 날짜 및 시간 → 자동 설정을 켜주세요. 또는 관리자에게 문의하면 강제 잠금 해제가 가능합니다.

Q. 앱이 업데이트되었다는 팝업이 떴어요.

[지금 업데이트] 버튼을 누르면 최신 버전으로 자동 전환됩니다. 업데이트 중 잠깐 화면이 새로고침됩니다. 나중에 버튼을 눌러도 무방합니다.

Q. 수료증은 언제 받을 수 있나요?

6개 미션이 모두 승인 완료된 후 [더보기] → [수료증 보기]에서 확인 가능합니다. 수료증 화면에서 이미지 저장 및 수료선물 신청이 가능합니다. 부활주일(4월 5일) 예배 때 선물을 드릴 예정입니다.

Technical

09. 기술 정보 (참고용)

🏗️ 구성

PWA (Progressive Web App) — 별도 앱스토어 설치 없이 웹 브라우저로 접속 후 홈화면에 추가하는 방식. iOS Safari, Android Chrome, 삼성 인터넷 모두 지원.

Firebase — Google의 클라우드 서비스. 익명 인증(학생), 이메일 인증(관리자), Firestore 데이터베이스 사용. 실시간 동기화.

Service Worker — 오프라인 캐싱. 인터넷이 끊겨도 기본 화면은 볼 수 있음. 앱 업데이트 시 자동 감지 후 팝업.

🔒 데이터 보안

학생 데이터는 Firebase Firestore에 저장되며 별도 서버 없이 Google 클라우드에서 보호됩니다. 학생 간 데이터는 서로 볼 수 없으며, 관리자만 전체 목록에 접근 가능합니다.

학생 앱은 회원가입 없는 익명 인증 방식이므로 개인정보 수집이 최소화됩니다 (이름·학년·그룹만 저장).

📱 지원 환경

iOS 14 이상 (Safari 권장) · Android 8 이상 (Chrome 또는 삼성 인터넷 권장) · PC 브라우저도 동작하나 모바일 최적화
